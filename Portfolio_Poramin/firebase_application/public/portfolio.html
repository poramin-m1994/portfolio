  • SENA SOLAR MONITORING FOR SENA 360 APPLICATION

    Solar is a real-time solar energy monitoring app that connects to each home's IoT device to display realtime power generation data. The app has two components:
    1. Resident-Facing App for monitoring and viewing power generation data, and
    2. Content Management System (CMS) for admins to manage device information for each Sena project.

    Project Responsibilities
    Third-Party API Data Validation and Integration: Verified and validated data received from third-party APIs, ensuring data accuracy, consistency, and compliance with internal data standards. Seamlessly integrated validated data into the system's database for further processing and analysis.

    User and Energy Data Database Design: Designed and implemented a robust database structure to efficiently store, manage, and retrieve user information and energy data. This included creating normalized tables, defining data relationships, and establishing data integrity constraints.
    Energy Data Visualization System Design: Architected a user-friendly data visualization system to effectively present energy data insights. This involved designing interactive charts, graphs, and dashboards to enable users to easily understand and analyze energy consumption patterns.
    Modular System Design for Future Product Integration: Employed a modular design approach to ensure the system's scalability and adaptability. This facilitated the integration of new products and features without disrupting existing functionalities, allowing for seamless data sharing and collaboration across the system.

  • Industrial Water Resource Management (IWRM)

    Iwrm is a comprehensive water quality monitoring system that records both raw and treated water quality parameters, allowing for real-time and historical data tracking (daily, monthly, yearly). It also records the status of related equipment such as pipelines, pumps, and water valves. The application comprises three components:
    1. Data Collection App: Field personnel use this app to record water quality and equipment data from various sources.
    2. Content Management System (CMS): Admins utilize this CMS to manage user information, generate reports, and edit water quality and equipment data.
    3. Real-Time Water Quality Dashboard: This dashboard displays real-time water quality data for on-screen monitoring.

    Project Responsibilities
    Comprehensive Database Design: Verified and validated data received from third-party APIs, ensuring data accuracy, consistency, and compliance with internal data standards. Seamlessly integrated validated data into the system's database for further processing and analysis.

    API Design for System Requirements: Designed every API endpoint for the project, ensuring seamless integration with system requirements and user needs. This involved defining API endpoints, specifying data formats, and implementing security measures
    Water Quality Reporting System Design: Created a comprehensive reporting system to generate detailed reports on water quality parameters. This involved designing report templates, integrating data visualization tools, and implementing data filtering capabilities
    Real-time Weather and Water Quality Dashboard API Design: Architected a RESTful API to provide realtime access to weather and water quality data. This involved designing API endpoints, caching mechanisms, and data retrieval algorithms.
